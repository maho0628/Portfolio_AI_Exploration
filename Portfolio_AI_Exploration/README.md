Manual Override
プレイヤー介入型AIを用いた探索アドベンチャー（Planning / Prototype）
概要

Manual Override は、
「AIに任せる行動」と「プレイヤーが操作で割り込む瞬間」の境界をテーマにした、
探索×操作介入型アドベンチャーゲームのプロトタイプです。

キャラクターは基本的に自動で行動しますが、
特定のタイミングでプレイヤー入力による介入が可能となる設計を採用しています。
本プロジェクトでは、AI設計そのものを主な検証対象としています。

制作目的

AI制御とプレイヤー操作が共存する構造の検証

行動処理と状態遷移を分離したAI設計の実装

後から調整・拡張しやすい、データ駆動型の構成を試す

「賢いAI」を作ることではなく、
**“扱いやすく、説明できるAI構造”**を目指しています。

ゲーム構成（予定）
探索パート

3Dフィールドを自由に移動

NavMesh を用いた簡易的な移動・巡回AI

敵やオブジェクトの発見によって戦闘へ遷移

戦闘パート

キャラクターは基本的に自動で行動

行動ゲージが溜まるとスキル準備状態に遷移

プレイヤー入力によりスキルを発動（または待機）

AI設計方針

行動ルールは全キャラクター共通

キャラクターごとの差異はデータで制御

行動処理と状態遷移を明確に分離

プレイヤー入力は「割り込みイベント」として扱う

AIは最適解を導く存在ではなく、
状態管理と操作受付を行う存在として設計しています。

使用技術

Engine: Unity 6

Language: C#

AI: Unity公式 Behavior Tree / Blackboard

Data: ScriptableObject
